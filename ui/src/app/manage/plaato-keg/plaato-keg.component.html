Device Setup Instructions

Step 1: Create new Device

  1. Enter a descriptive name
  2. Click the Submit button

Step 2: Put Device in Setup Mode

  1. Ensure your Plaato Keg is plugged into power
  2. Take the magnetic reset key (located on the bottom of the unit) and place it in the reset slot
  3. Reset the device by holding the magnet in the reset slot until the lights on the device start flashing, indicating it is in setup mode. (roughly 3-5 seconds)
  4. After reset, the device will create a WiFi access point (AP)
  5. Connect to the devices WiFi network (named something like "Plaato-XXXXX" or "PlaatoKeg-XXXXX" )

Step 3: Configure WiFi on Physical Device

DO NOT disconnect from the Plaato Keg device wifi
  1. Enter your WiFi SSID and password
  2. Click the Configure Device button
  3. A new window will open and you should see a message that looks like:
    {"status":"ok","msg":"Configuration saved"}
    Once verified the window should close automatically
  4. Once the device configurations is set and the device registers with the service, it should automatically disconnect you from the device Wifi
